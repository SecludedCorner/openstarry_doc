# archive/ — 過程紀錄層（Post-mortem Data Layer）

本目錄保存專案多團隊治理時代（2026-03 ~ 2026-05）的**過程殘留**：cycle 報告、投票紀錄、計數器登記、校準報告、研究方法論文件、已完結的實作計畫。共 158 檔，於 2026-06-12 蒸餾第一波依 [DISTILLATION_LIST.md](../DISTILLATION_LIST.md) 的逐檔判定移入，**原始內容與相對結構未動**（原路徑 `X/Y.md` → `archive/X/Y.md`）。

## 為什麼保留而不刪除

這批文件是 [RETROSPECTIVE.md](../RETROSPECTIVE.md) 所引用數據的**原始證據層**：87.5% spec-vs-merged gap、96% closure 灌水、十輪 210 票全數一致——這些數字的出處就在這裡的 cycle 紀錄中。對研究多智能體系統失效模式的人，這是罕見的完整田野資料；對其他讀者，它們是噪音——所以移到這裡，不擋導讀路徑。
